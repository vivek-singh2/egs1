## ✅ Features
✔️ Headings (`#`)  
✔️ Text Formatting (`*`, `**`, `~~`)  
✔️ Lists (`-`, `*`, `1.`)  
✔️ Tables (`| |`)  
✔️ Blockquotes (`>`)  
✔️ Code Blocks (```code```)  
✔️ Images (`![Alt Text](URL)`)  
✔️ Links (`[text](URL)`)  
✔️ Task Lists (`- [ ]`)  
✔️ Horizontal Lines (`---`)  
✔️ Emojis (`🚀`)
